# Веб-сервер для работы с amoCRM

## Развернутая версия

### https://amocrm-test-widget.herokuapp.com/

## Локальная установка

```
# Клонировать проект
git clone git@github.com:Evgen-Polyanskii/amoCRM.git

# Перейти в директорию проекта
cd amoCRM

# Установка
make setup

```

## Запуск

```
make start
```
